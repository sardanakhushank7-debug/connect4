Project Overview
This project is a console-based implementation of the classic Connect Four game. The game allows two players to take turns dropping discs into a 7x6 grid. The objective is to connect four discs horizontally, vertically, or diagonally before the opponent.

Game Rules
The board consists of 7 columns and 6 rows.
Players take turns selecting a column (1–7).
A disc is placed in the lowest available position in the selected column.
A player wins by connecting four of their discs in a row.
The game ends in a draw if the board is full and no player wins.
Planned Object-Oriented Design

Planned classes include:

GameController: Controls the game flow and turn management.
Board: Stores the game grid and contains logic for placing discs and checking win conditions.
Player: Represents a generic player.
HumanPlayer: Handles console input for human players.
ConsoleView: Displays the game board and messages in the console.
Development Plan
